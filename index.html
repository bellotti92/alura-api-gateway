Photo Collection API

A API Photo Collection contem os seguintes recursos:

Consulta pelo ID da photo. Metodo GET /photo/{id} 

            Exemplo:
            photo/1

Consulta pelo ASSUNTO da photo. Metodo GET /photo/assunto 

            Exemplo:
            photo/assunto?nome=batman

Consulta por ASSUNTO e COLECAO. Metodo GET /photo/consulta 

            Exemplo:
            /photo/consulta?assunto=batman&colecao=dc
